A modern iOS authentication app demonstrating **Swift concurrency** with **async/await**

## 📱 Features

- ✅ **Async/Await** - Modern Swift concurrency
- ✅ **Firebase Authentication** - Email/Password sign up & login
- ✅ **Input Validation** - Email format & password strength validation
- ✅ **MVVM Architecture** - Clean separation of concerns
- ✅ **Error Handling** - Comprehensive error management
- ✅ **UIKit Support** - Clean UI implementation
- ✅ **SwiftUI Ready** - Easily adaptable for SwiftUI
